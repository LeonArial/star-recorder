# 产品概述

星纪 ASR 服务端是一个独立的语音识别（Automatic Speech Recognition）服务，为星纪会议纪要平台提供实时和离线语音转写能力。

## 核心功能

- **实时语音识别**：通过 WebSocket 接收音频流，实时返回识别结果
- **离线文件转写**：支持上传音频文件进行批量转写
- **多模型融合**：
  - Paraformer：流式识别模型，快速出字
  - SenseVoice：复检模型，提高准确率
  - VAD：语音端点检测，自动分段
  - LLM 纠错：使用 Qwen3 进行结果优化
- **热词系统**：支持专有名词配置，提高特定领域识别准确率
- **时间戳生成**：提供句级时间戳，便于音频定位

## 技术特点

- 基于 FunASR 开源框架
- 支持多种音频格式（wav/mp3/ogg/flac/m4a/aac/wma/webm）
- 提供 REST API 和 WebSocket 两种接入方式
- 支持 CPU 和 GPU 加速
- 模型缓存机制，减少启动时间
- Docker 容器化部署

## 服务定位

本服务作为独立的 ASR 微服务，可被多个应用调用，当前主要为星纪会议纪要平台提供语音识别能力。
